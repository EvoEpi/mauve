Release 3.0.18

This release should be the last one before we move toward Cinema 3.1, and should be considered to be a "release candidate".

User Visible Changes

A few small usability changes, and some additions!
  • Lots of colour schemes added. Also color-convert.el for people who want to import more colour schemes.
  • Moved some of the menu items around.
  • Home directory is now selected by default for motif save
  • Resize keep square is default

Bug fixes

  • Stack trace dumping when gaps are removed from groups no longer occurs

Release-3.0.8

User Visible Changes

  • Can now choose format on motif save
  • Can save only a single motif, as well as all of them

  • Enhancements to the XML loading process exception handling, so problems are now a lot more informative
